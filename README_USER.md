# 사용자 가이드

> [메인 README](./README.md) | **사용자 가이드** | [개발자 가이드](./README_DEV.md)

경리 업무 리마인더를 처음 사용하시는 분을 위한 안내서입니다.

---

## 1. 시작하기

### 1-1. 회원가입

1. 웹 브라우저에서 `http://localhost:3000` 접속
2. **"회원가입"** 클릭
3. 이름, 이메일, 비밀번호 입력 → **"다음"**
4. 회사명 입력 (사업자등록번호는 선택) → **"회사 등록 및 시작"**

회원가입이 완료되면 자동으로 대시보드로 이동합니다.

### 1-2. 로그인

- 등록한 이메일과 비밀번호로 로그인
- 로그인 상태는 자동으로 유지됩니다 (7일간)

---

## 2. 대시보드

로그인 후 가장 먼저 보이는 화면입니다.

### 한눈에 보는 요약

화면 상단에 4개의 카드가 표시됩니다:

| 카드 | 의미 |
|------|------|
| **오늘 마감** | 오늘이 마감일인 일정 수 |
| **지연** | 마감일이 지났지만 완료하지 않은 일정 수 |
| **예정** | 아직 마감일이 오지 않은 일정 수 |
| **완료** | 이미 처리 완료한 일정 수 |

### 일정 카드

각 일정은 카드 형태로 표시됩니다:

- **체크박스**: 클릭하면 완료/미완료 전환
- **D-Day 배지**: D-3(3일 남음), D-Day(오늘), D+2(2일 지남)
- **색상 의미**:
  - 빨간색 왼쪽 테두리 = 지연된 일정
  - 주황색 = 오늘 마감
  - 파란색 = 여유 있는 일정
- **"조정됨" 태그**: 원래 마감일이 공휴일/주말이어서 자동 조정된 일정

### 필터링

- **카테고리**: 원천세, 4대보험, 부가세, 급여 등으로 필터
- **상태**: 전체 / 미완료 / 완료

### Excel 연동

- **내보내기** (다운로드 아이콘): 현재 일정을 Excel 파일로 다운로드
- **가져오기** (업로드 아이콘): Excel 파일에서 일정을 일괄 등록

## 3. 템플릿 적용 (핵심 기능)

**템플릿 하나로 연간 일정을 자동 생성**하는 핵심 기능입니다.

### 사용 방법

1. 왼쪽 메뉴에서 **"템플릿"** 클릭
2. **"템플릿 적용"** 버튼 클릭
3. 원하는 템플릿 선택:
   - **법정신고일** — 원천세, 4대보험, 부가세, 법인세 등 (33건)
   - **급여 처리** — 급여 지급일, 급여 확정일 (24건)
   - **HR 업무** — 4대보험 취득/상실, 연차 점검 등 (17건)
4. **연도** 선택 (예: 2026)
5. **"미리보기"** 클릭 → 생성될 일정 목록 확인
6. **"적용"** 클릭 → 일정이 일괄 생성됨

### 자동 조정 예시

| 원래 마감일 | 이유 | 조정된 마감일 |
|-------------|------|---------------|
| 2026-01-01 (목) | 신정 공휴일 | 2026-01-02 (금) |
| 2026-03-01 (일) | 삼일절 + 일요일 | 2026-03-02 (월) |
| 2026-10-03 (토) | 개천절 + 토요일 | 2026-10-05 (월) |

설날, 추석 등 음력 공휴일도 자동 계산됩니다.

## 4. 캘린더

월간 캘린더 형태로 일정을 한눈에 볼 수 있습니다.

- **좌우 화살표**: 이전/다음 월 이동
- 각 날짜 칸에 해당일의 일정이 표시됨
- 일정이 3개 이상이면 "+N건"으로 요약 표시
- 일요일은 빨간색, 토요일은 파란색

## 5. 팀 협업

### 멤버 초대

회사 소유자(Owner) 또는 관리자(Admin)가 팀원을 초대할 수 있습니다.

역할별 권한:

| 역할 | 일정 조회 | 일정 편집 | 멤버 관리 |
|------|----------|----------|----------|
| Owner | O | O | O |
| Admin | O | O | O |
| Member | O | O | X |
| Viewer | O | X | X |

### 실시간 동기화

- 팀원 A가 일정을 수정하면, 팀원 B의 화면에 **즉시 반영**
- 별도의 새로고침 불필요

## 6. 알림

### D-Day 알림

대시보드에서 각 일정의 D-Day를 실시간 확인:
- **D-7** ~ **D-1**: 마감 임박 (주황색)
- **D-Day**: 오늘 마감 (빨간색)
- **D+N**: 지연됨 (빨간색)

### 알림 요약 API

앱에서 자동으로 알림 요약을 조회합니다:
- 오늘 마감 일정
- 지연된 일정
- 7일 이내 마감 일정

## 7. 모바일 사용 (PWA)

별도 앱 설치 없이 모바일 브라우저에서 사용할 수 있습니다.

### 홈 화면에 추가

1. 모바일 브라우저에서 `http://your-server:3000` 접속
2. **브라우저 메뉴 → "홈 화면에 추가"**
3. 앱처럼 아이콘이 생성됩니다

### 오프라인 지원

- 한 번 접속한 페이지는 오프라인에서도 볼 수 있습니다
- 네트워크 복구 시 자동 동기화

---

## 8. 데스크톱 앱

Electron 기반 데스크톱 앱으로도 사용할 수 있습니다.

- **시스템 트레이**: 창을 닫아도 트레이에서 계속 실행
- **알림**: 시스템 알림으로 마감 임박 일정 안내
- **싱글 인스턴스**: 앱이 중복 실행되지 않음

## FAQ

**Q. 공휴일 정보는 정확한가요?**
A. 한국천문연구원의 음력 데이터를 기반으로 설날·추석·부처님오신날을 계산하며, 2021년부터 적용된 대체공휴일 확대 법안도 반영되어 있습니다.

**Q. 여러 회사의 일정을 관리할 수 있나요?**
A. 네, 한 계정으로 여러 회사를 등록하고 전환하며 사용할 수 있습니다.

**Q. 데이터는 안전한가요?**
A. 비밀번호는 bcrypt로 해싱되고, API 통신은 JWT 토큰으로 보호됩니다. 환경변수에 민감 정보를 저장하여 코드에 노출되지 않습니다.
